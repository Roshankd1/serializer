invokeMethod

fun <R> Any.invokeMethod(name: String, vararg params: Any?): R

Invoke a specific method (not only declared) of an object through reflection No matter if it's not accessible

Parameters

R - return type of the method

name - of the method

params - arguments of the method (must be compatible with the length and the types of the method)

Exceptions

NoSuchMethodException - if a matching method is not found.

InvocationTargetException - if the underlying method throws an exception.

Author
sliep

Receiver
the instance that provide the method or it's declaring class to invoke a static method

Return
the result of method
