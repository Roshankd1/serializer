callSetter

fun Any.callSetter(fieldName: String, value: Any?): Unit

Call kotlin (or java) property setters of a receiver object through reflection No matter if it's not accessible If the property name is 'foo', 'setFoo()' method will be called (if exists)

Parameters

fieldName - of the property

value - new value for the property

Exceptions

NoSuchMethodException - if a matching method is not found.

InvocationTargetException - if the underlying setter throws an exception.

Author
sliep

Receiver
the instance having the magic property
