open fun log(depth: Int = -1, message: () -> Any?): Unit

Log a message

Parameters

depth - a positive number included 0. every unit of depth correspond to 4 spaces before the message when printing the log to indent the data. default is -1 it means that will be taken from the 'depth' static variable in the class or zero if not defined

message - a function that will be executed only if 'LOG' is true. the result of this call must be the message to print or null to print nothing
