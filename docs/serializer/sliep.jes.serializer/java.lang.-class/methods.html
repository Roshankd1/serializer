fun Class<*>.methods(modifiers: Int = 0, excludeModifiers: Int = 0): Array<Method>

Get all methods (not only declared) of the receiver class matching given parameters

Parameters

modifiers - modifiers that a method should have to be included or 0 to include all except excluded. default is include all

excludeModifiers - modifiers that a method should not have to be included or 0 to exclude none. default is exclude none

Author
sliep

Receiver
the declaring Class of the methods

Return
an array of found matching methods
