fun <T> Class<T>.constructors(modifiers: Int = 0, excludeModifiers: Int = 0): Array<Constructor<out T>>

Get all declared constructor of the receiver class matching given parameters

Parameters

T - Type of the class

modifiers - modifiers that a constructor should have to be included or 0 to include all except excluded. default is include all

excludeModifiers - modifiers that a constructor should not have to be included or 0 to exclude none. default is exclude none

Author
sliep

Receiver
the declaring Class of the constructors

Return
an array of found matching constructors
