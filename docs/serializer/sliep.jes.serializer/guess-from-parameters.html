guessFromParameters

fun <M : Executable> guessFromParameters(clazzName: String, members: Array<out M>, name: String?, params: Array<out Any?>): M

Makes possible to find a compatible method having only it's parameter values and the name

Parameters

clazzName - only for debug: the name of declaring class

members - all methods in that class

name - of the method

params - input method arguments (null arguments are jolly, this can lead to ambiguity problems)

Exceptions

NoSuchMethodException - if a matching method is not found.

Author
sliep

Return
the matching method
